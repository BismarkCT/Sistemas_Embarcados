O Raspbian não tem a comunicação I2C
habilitada inicialmente. Para utiliza-la,
você deve habilita-la da seguinte maneira:

1 - Execute

	$ sudo raspi-config

vá em Advanced options -> I2C, e habilite o acesso:

	Would you like the ARM I2C interface to be enabled? Yes.

2 - Execute

	$ cat /boot/config.txt | grep i2c

Deverá aparecer o texto

	dtparam=i2c_arm=on

indicando que a I2C está habilitada.

3 - Reinicie (reboot) o sistema para tornar esta mudança efetiva.
